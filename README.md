

# 가상 환경 준비 

```
conda create -n fastapi python=3.10 -y

conda activate fastapi
```


```
sudo apt install python3-venv

python3 -m venv venv 

source venv/bin/activate
```


# 패키지 설치 

```
sudo apt update
sudo apt install ffmpeg
```

```
pip install -r requirements.txt
```


# 한국어 폰트 다운 

```
sudo apt-get install fonts-nanum
```


# fastapi 웹 실행

```
source venv/bin/activate
python run.py
```

`http://localhost:8000/`

로 접근해야함
